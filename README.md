### 📝 SyncDoc — AI-Powered Collaborative Workspace
<div align="center">

[![Live Demo](https://img.shields.io/badge/🔴%20LIVE-syncdocme.netlify.app-6C3483?style=for-the-badge&logo=netlify)](https://syncdocme.netlify.app/)
[![Assignment](https://img.shields.io/badge/Regrip%20India-Full%20Stack%20Assignment-FF4500?style=for-the-badge)](https://syncdocme.netlify.app/)

</div>

> **Real-time collaborative document workspace** (Notion/Google Docs style). Multiple users can edit simultaneously with conflict-free sync, live chat, file sharing, and Gemini AI assistance — all in 48 hours.

**Key Features:**
- ✏️ **Real-Time Rich Text Editor** — simultaneous multi-user editing, changes sync instantly via WebSockets
- 👥 **Presence & Live Cursors** — see who's editing which paragraph in real time
- 🔄 **Conflict Resolution** — OT/CRDT-inspired race condition handling — no data overwrite
- 💬 **Live Chat Sidebar** — document-scoped real-time chat with instant message delivery
- 📁 **File Sharing** — upload images/PDFs directly in chat stream, stored via cloud storage
- 🔐 **RBAC Security** — Owner / Editor / Viewer roles enforced at backend (403 Forbidden for violations)
- 🤖 **Gemini AI Panel** — "Summarize Document" & "Fix Grammar & Tone" with streaming responses
- ⚡ **Optimistic UI** — instant UI updates while backend processes (rename, send message, etc.)
